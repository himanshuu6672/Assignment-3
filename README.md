# Python Math Programs

This repository contains two Python scripts:

## Task 1

-   Calculates the factorial of a number using recursion.

## Task 2

-   Uses the `math` module to calculate:
    -   Square root
    -   Logarithm
    -   Sine

## How to Run

``` bash
python Task\ 1.py
python Task\ 2.py
```

## Requirements

-   Python 3.x
